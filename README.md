****Student Management System (C++)****

**Overview**

A simple command-line Student Management System written in C++.
The program allows users to store and manage student records including name, roll number, and marks.

This project demonstrates basic object-oriented programming concepts and use of STL containers.

**Features**

--Add a student

--Display all students

--Search student by roll number

--Simple menu-driven interface


Concepts Used

--Classes and Objects

--Encapsulation

--Vectors (std::vector)

--Functions

--Basic input/output handling


**Program Structure**

main.cpp

│

├── Student class

│   ├── input()

│   ├── display()

│   └── getRollNumber()

│

├── addStudent()

├── displayStudents()

└── searchStudent()


**Example Menu**

1. Add Student
2. Display Students
3. Search Student
4. Exit
